## Installation
1. Clone the repo\
`git clone https://github.com/malin-nilsson/Mindfully`

2. Install npm packages\
`npm i`

3. CD into repo\
`cd mindfully`

5. Run React \
`npm start`

## Testing
1. Make sure React is running, open a new terminal and start Cypress\
`npx cypress open`
